
# Recipe Food Search

A React application to search and discover recipes using the API.

---

## Requirements

- Node.js v17+

---

## Quick Start

```bash
npm install
npm start
```

Open **http://localhost:3000** in your browser.

---

## Features

- Search recipes by name or ingredient
- Browse recipes by category
- View detailed recipe instructions and ingredients
- Responsive layout built with React Bootstrap

---

## Tech Stack

| Technology | Purpose |
|---|---|
| React | UI framework |
| React Bootstrap | Component library |

---


## Design Decisions

The app uses a single page layout with a search bar at the top and recipe cards below. React Bootstrap was chosen for its responsive grid system — recipe cards adapt from one column on mobile to four columns on desktop without custom CSS.
